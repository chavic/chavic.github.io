Lunas

A private preserving contact tracing app that is decentralized. This project consists of an app and a graphQL api that is hosted on google cloud platform. Lunas consist of an app, a graphQL api and postgresSQL database that. The api and database are hosted on google cloud using cloud datebase and cloud run.
